Våre plattformer

HTML5

Mobile webapplikasjoner i moderne nettlesere kan bruke geo-funksjonalitet, lagre lokalt og på mange måter fremstå som en app. Det er god grunn til å tro at fremtidens apps er uavhengig av produsenter, og bygget på HTML/CSS/JS. Les mer om HTML5››

Native (Objective-C + Java)

For optimal ytelse og umiddelbar tilgang til siste funksjonalitet fra Android/iOS SDK, trenger man en native applikasjon. Vi gjør utvikling i både Objective-C (iOS) og Java (Android), og kan derfor levere store og krevende applikasjoner. Les mer ››

PhoneGap / Cordova

Med applikasjonen utviklet i HTML-baserte PhoneGap / Cordova får man tilgang til viktige applikasjoner som kamera, aksellerometer og GPS via javascript, men med kort vei til markedet på flere plattformer samtidig. Les mer om PhoneGap ››

Appcellerator Titanium

Titanium er en javascriptbasert løsning som gir deg native brukergrensesnitt for både iOS og Android, samt tilgang til det aller meste av annen funksjonalitet du finner i iOS og Android. Plattformen har blitt svært populær. Les mer om Titanium ››
